Tính năng mới: Gần đây Git (2.47–2.49) bổ sung nhiều tính năng đáng chú ý: hỗ trợ build bằng Meson bên cạnh Make/CMake, tăng tốc xử lý SHA-1 và đưa --remerge-diff vào range-diff để review dễ đọc hơn; tiếp tục hoàn thiện backend lưu trữ ref “reftable” và cải tiến git maintenance; thử nghiệm incremental multi-pack indexes giúp quản lý nhiều packfile hiệu quả trên repo lớn; giới thiệu name-hash v2 để đóng gói nhanh hơn trong pack-objects; thêm lệnh git backfill để “điền bù” blob lịch sử còn thiếu và dạy git clone tạo shallow clone từ một commit không ở đầu nhánh; đồng thời chuẩn bị hỗ trợ zlib-ng và mở đường giao diện ngoại ngữ cho Rust trong mã nguồn. Lưu ý bảo mật: năm 2025 có vá lỗi nghiêm trọng liên quan submodule, nên cập nhật lên bản Git đã vá (như 2.47.3, 2.48.2, 2.49.1…).
